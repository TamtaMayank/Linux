### 🔍 `grep` – Search Text in Files
```bash
grep "text" file.txt           # Find lines containing "text"
grep -i "text" file.txt        # Case-insensitive search
grep -r "main" folder/         # Recursive search
grep -n "word" file.txt        # Show line numbers
